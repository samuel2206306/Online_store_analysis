# Online Store Order Analysis

This project demonstrates how to analyze sales data using **PostgreSQL** and **Excel**.

## Features
- Query orders, customers, products to calculate total sales per customer.
- Analyze number of orders per month.
- Export query results to CSV for Excel analysis.
- Build pivot tables and charts to visualize monthly sales trends and top-selling products.

## Technology
- SQL (PostgreSQL)
- Excel (Pivot Tables, Charts)

## How to Use
1. Run `database_setup.sql` to create tables and insert sample data.
2. Run queries in `queries.sql` to extract order details.
3. Export results to CSV and open in Excel to create pivot tables and charts.
